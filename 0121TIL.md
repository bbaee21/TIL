# 함수 중요

## 특징

* return 과 print의 차이점
  * return : 최종 결과 값 반환
  * print : 단순 값 화면에 출력
* 함수명, 인자, 매개변수
  * 함수명 -> 함수이름
  * 인자 : 실제 입력되는 값
  * 매개변수 : 값을 받을 변수

* 키워드와 위치인자
  * 키워드, 위치인자 혼용 시 위치인자 우선, 그다음 위치 맞게 하면 사용 가능 / 키워드 인자 먼저 넣고 위치 인자 넣을 시 사용 불가
* 가변인자는 변수명 앞에 *

* SQL JOIN



* Inner / outer

  * inner

    * 쉽게 생각해서 교집합
    * A, B 모두 데이터 존재할 때 두 가지 테이블 조인할 때 사용한다.

    

  * outer (LEFT / RIGHT)

    * 각 각 집합, 합집합 
    * 한 쪽만 데이터 존재할 때 두 가지 테이블 조인할 때 사용함
    * LEFT는 왼쪽 테이블 기준으로 조인함 RIGHT는 반대로 생각하면 됨
    * LEFT OUTER JOIN LEFT JOIN / RIGHT OUTER JOIN RIGHT JOIN 동일하게 작동함
